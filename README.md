# Machine-Learning-

This is a continuously updated repository that documents personal journey on learning data science, machine learning related topics.

**Goal:** Introduce machine learning contents in Jupyter Notebook format. The content aims to strike a good balance between mathematical notations, educational implementation from scratch using Python's scientific stack including numpy, scipy, pandas, matplotlib, etc. and open-source library usage such as scikit-learn, keras, tensorflow, etc.
